# Azure Home Lab
Personal lab environment at TCS as a system administrator
Goal: simulate an enterprise IT environment using Microsoft Azure.

* Active Directory Domain Services
- Deployed AD DS on Azure VM
- Created OUs, user accounts, and group policies (GPO)
- Simulated enterprise user lifecycle management

* Entra ID (Azure AD)
- Configured Entra ID tenant
- Set up Conditional Access policies
- Enabled MFA for all user accounts
- Configured RBAC roles for least-privilege access

* Intune MDM
- Set up Intune for endpoint management
- Configured device compliance policies
- Enrolled test devices and applied security baselines

* WSUS Patch Management
- Deployed WSUS server on Azure VM
- Configured patch approval workflows
- Simulated enterprise endpoint patching cycles

* Architecture
- Platform: Microsoft Azure
- Identity: Entra ID + Active Directory DS
- Endpoint Management: Microsoft Intune
- Patching: WSUS
- Monitoring: Azure Monitor

* Purpose
Built to maintain hands-on skills with enterprise Microsoft infrastructure
while completing graduate studies — directly mirrors production environments
managed at Tata Consultancy Services.

* Certifications
- Microsoft Azure Fundamentals (AZ-900)
- CompTIA A+ (in progress)
